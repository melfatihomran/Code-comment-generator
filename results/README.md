# Results

Populate this folder after running `notebooks/codet5_finetune_codesearchnet.ipynb` on Kaggle:

- `test_metrics_baseline.json` — pretrained codet5-small, no fine-tuning
- `test_metrics_finetuned.json` — your fine-tuned model
- `qualitative_samples.md` — side-by-side code / predicted comment / reference comment

Then copy the numbers into the Results table in the top-level `README.md`.
